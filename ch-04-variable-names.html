Chapter 4: Variable ke Naam Rakhne ke Rules

  • Home

Rules ka Khulasa:

  1. Space Nahi: Naam mein space nahi ho sakta.
  2. Allowed Characters: Sirf English letters, numbers, underscore _, aur dollar sign $ istemal karen (lekin number se shuru na karein).
  3. camelCase: Do ya zyada lafzon ke naam ke liye camelCase istemal karein (e.g., myVariableName). Yeh professional tareeqa hai.
  4. Case-Sensitive: Name aur name do alag variables hain.
  5. Meaningful Names: Naam hamesha matlab wala rakhein. x ke bajaye customerName behtar hai.

In rules ko follow karne se aapka code saaf, ghaltiyon se paak, aur parhne mein aasan rehta hai.
