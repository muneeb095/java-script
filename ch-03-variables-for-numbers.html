Chapter 3: Variables for Numbers

  • Home

Is chapter mein hum seekhenge ke numbers ko variables mein kaise store karte hain.

Sabse Bada Farq: Quotation Marks "" Nahi!

Jab aap number store karte hain, to aap quotation marks "" ka istemal nahi karte.

var weight = 150;

Iska faida yeh hai ke aap in par math (hisaab-kitaab) kar sakte hain!

Misal: Variable ke saath Math karna

var originalNum = 23;
var newNum = originalNum + 7;

Is code ke baad, newNum variable ke andar 30 store ho jayega.

Zaroori Rule:

Variable ka naam kabhi bhi number se shuru nahi ho sakta.

  • var 1stPlace = "Gold"; ❌ (Ghalat)
  • var place1 = "Gold"; ✅ (Sahi)
