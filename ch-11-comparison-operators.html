Chapter 11: Comparison Operators (Compare karne wale Nishan)

  • Home

Yeh chapter if statement ki asal taqat ko unlock karta hai. Hum yahan woh saare nishan (operators) seekhenge jinke zariye hum do cheezon ko compare kar sakte hain. In operators ka jawab hamesha true (sach) ya false (jhoot) mein aata hai.

1. `===` (Equality Operator)

Matlab: Kya yeh barabar hai?

Yeh check karta hai ke kya do cheezein bilkul ek jaisi hain ya nahi. Yeh value aur type dono check karta hai.

if (userAnswer === "Vatican") { ... }

Case-Sensitive: Text (strings) ko compare karte waqt, yeh case-sensitive hota hai. Iske liye "Rose" aur "rose" do alag cheezein hain.

2. `!==` (Not-Equal Operator)

Matlab: Kya yeh barabar NAHI hai?

Yeh === ka bilkul ult (opposite) hai. Yeh check karta hai ke kya do cheezein aapas mein barabar nahi hain.

if (userTicket !== 487208) {
    alert("Sorry, you did not win.");
}

3. `>` (Greater Than Operator)

Matlab: Kya yeh bara hai?

Yeh aam taur par numbers ke liye istemal hota hai.

if (age > 18) { ... } // Agar umar 18 se bari hai

4. `<` (Less Than Operator)

Matlab: Kya yeh chota hai?

if (temperature < 0) { ... } // Agar temperature 0 se kam hai

5. `>=` (Greater Than or Equal To Operator)

Matlab: Kya yeh bara hai ya barabar hai?

if (score >= 50) { ... } // Agar score 50 ya us se zyada hai

6. `<=` (Less Than or Equal To Operator)

Matlab: Kya yeh chota hai ya barabar hai?

if (price <= 100) { ... } // Agar qeemat 100 ya us se kam hai

Advance Topic: Logical Operators (Conditions ko Jorna)

Aap if statement mein ek se zyada conditions ko bhi ek saath check kar sakte hain. Iske liye "Logical Operators" istemal hotay hain. Yeh aage ke chapters mein ayenge, lekin yahan inka ta'aruf zaroori hai.

1. `&&` (AND Operator - "Aur")

Yeh tab true hota hai jab iske dono taraf ki conditions sach hon. Agar ek bhi condition ghalat hui, to yeh false ho jayega.

if (age >= 18 && hasIDCard === true) {
    alert("Aap vote daal sakte hain.");
}

Iska matlab hai: "Agar umar 18 ya usse zyada hai AUR ID card bhi hai, tab hi alert dikhao."

2. `||` (OR Operator - "Ya")

Yeh tab true ho jata hai jab iske dono taraf ki conditions mein se koi ek bhi sach ho. Yeh sirf tab false hota hai jab dono conditions ghalat hon.

if (day === "Saturday" || day === "Sunday") {
    alert("It's a weekend!");
}

Iska matlab hai: "Agar din Saturday hai YA Sunday hai, to alert dikhao."
