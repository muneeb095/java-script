Chapter 2: Variables for Strings

  • Home

Variable ek container ki tarah hai jismein aap koi information (jaise text) store kar sakte hain aur us container ko ek naam de dete hain. Jab bhi aapko woh information chahiye hoti hai, aap sirf us container ka naam istemal karte hain.

Aasan Steps mein samjhein:

  1. Variable Banana (Declare karna): var keyword ka istemal karein.
  2. Variable ko Naam Dena: var ke baad usay ek naam dein (e.g., myName).
  3. Information Store karna (Assign karna): Naam ke baad = ka nishan lagayein.
  4. Value Likhna: Text (string) ko hamesha quotation marks "" mein likhein.

Misal 1: Naam store karna

var name = "Mark";

Ab jahan bhi aap name likhenge, JavaScript uski jagah "Mark" samjhega.

Misal 2: Variable ko `alert` ke saath istemal karna

var message = "Thanks for your input!";
alert(message);

Yahan alert ke andar message ko quotes ke baghair likha gaya hai, kyunki ab yeh ek variable hai.
