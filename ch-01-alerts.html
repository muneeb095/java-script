Chapter 1: Alerts (Pop-up Messages)

  • Home

JavaScript mein, alert ek chota sa pop-up box hota hai jo aapke web page par user ko koi message dikhane ke liye istemal hota hai. Jaise, agar aap user ko "Thank you" kehna chahte hain, to ek alert box achanak screen par aayega aur us par aapka message likha hoga.

Aasan Steps mein samjhein:

  1. alert Keyword: Alert box banane ke liye, aap alert likhte hain. Yeh JavaScript ka ek khaas lafz (keyword) hai, isliye iska "a" hamesha chota (lowercase) hona chahiye.
  2. Parentheses (): alert ke baad, aap hamesha parentheses () lagate hain. Inke andar aap message likhte hain.
  3. Quotation Marks "": Message ko hamesha quotation marks "" (double quotes) ke andar likha jaata hai.
  4. Semicolon ;: Har JavaScript command ke akhir mein, ek semicolon ; lagana ek achi practice hai.

Misal (Example):

Agar aap user ko "Thanks for your input!" ka message dikhana chahte hain:

alert("Thanks for your input!");

Yeh code kya karega? Browser screen par ek pop-up box dikhayega jis par "Thanks for your input!" likha hoga.
