Chapter 1: Alerts (Pop-up Messages)

  • User ko message dikhane ke liye alert("Your Message"); istemal hota hai.

Chapter 2 & 3: Variables (Strings & Numbers)

  • Variable banane ke liye var istemal hota hai.
  • Text (string) hamesha quotes "" mein hota hai: var name = "Ali";
  • Number hamesha bina quotes ke hota hai: var age = 25;

Chapter 4: Variable Naming Rules

  • Naam mein space nahi ho sakta.
  • Naam number se shuru nahi ho sakta.
  • Naam ke liye camelCase (e.g., myVarName) istemal karein.
  • JavaScript case-sensitive hai (name aur Name alag hain).

Chapter 5 & 7: Math Expressions

  • Aam operators: +, -, * (multiply), / (divide).
  • % (modulus) division ke baad baaqi (remainder) batata hai.
  • Calculation ke order ko control karne ke liye hamesha parentheses () istemal karein.

Chapter 6: Increment (++) & Decrement (--)

  • ++ variable mein 1 jama karta hai.
  • -- variable se 1 minus karta hai.
  • Postfix (num++): Pehle purani value istemal hoti hai, phir value barhti hai.
  • Prefix (++num): Pehle value barhti hai, phir nayi value istemal hoti hai.

Chapter 8: Concatenating Text Strings (Jorna)

  • Text (strings) ko jorne ke liye + istemal hota hai.
  • Important: Agar + ke saath ek string aur ek number ho, to number bhi string ban kar jurr jaata hai (e.g., "Result: " + 2 + 2 banega "Result: 22").

Chapter 9: Prompts (User Input)

  • User se input lene ke liye prompt("Sawal?", "Default Jawab"); istemal hota hai.
  • Prompt se milne wala jawab hamesha string (text) hota hai.

Chapter 10: `if` Statements (Faisla Karna)

  • Faisla karne ke liye: if (condition) { ... }.
  • Shart (condition) () ke andar likhi jaati hai.
  • Value check (compare) karne ke liye hamesha === (triple equals) istemal karein. (Single = value assign karne ke liye hota hai).
