1. Increment Operator: ++ (Ek Barhanay wala)

Yeh kisi variable mein 1 jama karne ka shortcut hai. num = num + 1; ki jagah num++; likhein.

2. Decrement Operator: -- (Ek Kam Karne wala)

Yeh kisi variable se 1 minus karne ka shortcut hai. num = num - 1; ki jagah num--; likhein.

Nishan Pehle ya Baad Mein? (Prefix vs Postfix)

num++ (Postfix - Nishan Variable ke Baad)

Usool: "Pehle istemal karo, phir barhao."
var num = 1;
var newNum = num++;
Iska matlab hai pehle num ki purani value newNum mein daalo, phir num ki value ko barha do.

++num (Prefix - Nishan Variable se Pehle)

Usool: "Pehle barhao, phir istemal karo."
var num = 1;
var newNum = ++num;
Iska matlab hai pehle num ki value barhao, phir us nayi value ko newNum mein daalo.
