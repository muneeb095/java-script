Chapter 8: Text ko Jorna (Concatenation)

Text (strings) ko aapas mein jorne ke liye + ka nishan istemal hota hai.

Misal:

var userName = "Susan";
alert("Thanks, " + userName + "!");

Result: Alert box mein "Thanks, Susan!" likha aayega.

Zaroori Concept: Number aur Text ko Jorna

Agar aap + ke saath ek string aur number istemal karenge, to JavaScript number ko bhi string bana kar jor dega, jama (add) nahi karega.

Misal:

alert("2 plus 2 equals " + 2 + 2);

Sahi Result: "2 plus 2 equals 22"
